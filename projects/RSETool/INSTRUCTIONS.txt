A. CSV Maker
	- This tool works by pulling TABLE data values from a Word OR Excel document.
	1. FOR WORD DOCS:
	   The source file/document *MUST* have: 
		a. the contact information contained within a TABLE
		b. the TABLE must have headers (i.e. "first name", "email", etc.). The order of the values does not matter.
	2. FOR EXCEL DOCS:
	   The source file/document *MUST* have:
		a. headers (i.e. "first name", "email", etc.). The order of the values does not matter.
	
B. Payload Picker
**** THE TOOL WILL NOT FUNCTION PROPERLY IF THE TWO PAYLOAD FOLDERS ARE MISSING ****
	- This will let you copy to clipboard the html code of one of the selected email template
	- The code copied will change appropriately depending on whether you select "Insight" or "KnowBe4"

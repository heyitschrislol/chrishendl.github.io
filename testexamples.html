DEPRECIATION CALCULATOR
A PROGRAM TO CALCULATE DEPRECIATION
  • The program begins calculation upon clicking either the "Get Straight Line" or "Get Double Declining" buttons.
  • Values are pulled from the text-boxes and stored as variables to be passed into one of two methods (depending on which button is pressed).
  • Within the method, the program loops through each year of the inputted "Use Life" and calculates both yearly and total depreciation values using the parameter values that were passed. Additionally, each loop within the method creates a new table row to store its' yearly calculations and appends the new row to a data table.
  • The calculated data are displayed to the user within a data table object, divided neatly into columns and rows in ascending order beginning with the first year of depreciation.
